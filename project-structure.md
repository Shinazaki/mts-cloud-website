# Документация по структуре проекта MTS Cloud Portal

## Описание
Данный проект представляет собой Frontend-часть веб-приложения для облачного сервиса (аренда серверов, биллинг, мониторинг и т.д.). Интерфейс реализован в соответствии с предоставленным дизайном.

## Технологический стек
- **Вью/Фреймворк:** React 18
- **Сборщик:** Vite
- **Язык:** TypeScript
- **Маршрутизация:** React Router DOM (v6)
- **HTTP-клиент:** Axios (для интеграции с `spacehack-mts-cloud-backend`)
- **Стилизация:** Чистый CSS с использованием Custom Properties (переменных) для дизайн-системы.
- **Иконки:** Material Symbols Outlined (Google Fonts)
- **Шрифты:** Nunito

## Архитектура папок (`src/`)

- `api/` — Содержит `client.ts` с конфигурацией Axios и методами для работы с бэкендом (auth, servers, billing).
- `components/` — Переиспользуемые компоненты:
  - `Header/` — Верхняя панель пользователя (баланс, аккаунт).
  - `Sidebar/` — Левое навигационное меню.
- `layouts/` — Обёртки для страниц:
  - `DashboardLayout.tsx` — Основной макет приложения (включает Sidebar и Header).
  - `AuthLayout.tsx` — Макет для страниц авторизации/регистрации с характерным фоном.
- `pages/` — Компоненты-страницы, привязанные к маршрутам:
  - `Auth/` — `Login.tsx`, `Register.tsx` (Формы входа).
  - `Servers/` — `Servers.tsx` (Список серверов, поиск, таблица данных).
  - `Billing/` — `Billing.tsx` (Счета, история, настройки - с переключением вкладок).
  - `Settings/` — `Settings.tsx` (Настройки профиля, включая переключение языка RU/EN).
  - `SecondaryPages/` — Заглушки для остальных страниц (Бэкапы, Мониторинг, Трафик, Поддержка, API, Q&A, Что нового).
- `App.tsx` — Корневой компонент с настройкой маршрутизации (`BrowserRouter`).
- `main.tsx` — Точка входа React-приложения.
- `index.css` — Глобальные стили, сброс (reset) и переменные цветов (`--c-red`, `--c-dark-blue` и т.д.).

## Дизайн-система и CSS
Стилизация построена на БЭМ-подобной методологии с использованием CSS-переменных, что позволяет легко менять цветовую схему.
Основные цвета (согласно требованиям):
- `#780000` (Dark Red - Sidebar, формы Auth)
- `#C1121F` (Red - Кнопки, элементы таблиц, бордеры)
- `#E30613` (Bright Red - Акценты)
- `#003049` (Dark Blue - Текст, заголовки)
- `#FFFFFF` (White - Фоны контентных блоков)

## Интеграция с Backend
Клиент axios настроен в `src/api/client.ts`. На данный момент используется Interceptor для добавления Bearer-токена в заголовки запросов. Для полноценной интеграции необходимо заменить `BASE_URL` на актуальный адрес бэкенда (`spacehack-mts-cloud-backend`) и подключить методы из объекта `api` внутри компонентов (через `useEffect` или библиотеки типа React Query).
